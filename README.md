# VideoNote Server

## Tech stack

- [graphql](https://graphql.org)
- [mongoose](https://mongoosejs.com)
- [babel](https://babeljs.io)
- [express](https://expressjs.com)

### Run it

Create a `.env` at project root specifying your **MONGODB_URI** (refer to _.env.example_) and then...

`npm start`
